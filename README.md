# Auto-Push Script

---

## Description
This script automates the process of adding, committing, and pushing changes to a GitHub repository. It simplifies version control workflows by executing standard Git commands in a single sequence.

---

## Features
- Automatically stages all changes (git add .)

- Commits changes with a timestamp or custom message

- Pushes updates to the remote repository

 ## Prerequisites
- Git installed and configured

- Valid credentials or SSH keys for the target repository

## Installation
Clone the repository:

```bash
git clone https://github.com/Mrdolls/Auto-Push-Script.git
```

Navigate to the directory:
```bash
cd Auto-Push-Script
```
Ensure the script has execution permissions:
```bash
chmod +x auto-push.sh
```

## Usage
Run the script from your terminal:
```bash
./auto-push.sh
```
